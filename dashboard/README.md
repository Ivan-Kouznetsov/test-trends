# dashboard

React + Vite frontend for exploring failure trends from analyzed test data.

## What this module does
- Displays totals, failure rate, and latency summaries.
- Shows failure-rate-over-time line chart.
- Provides endpoint/test/concurrency/status/error filters.
- Renders top failing endpoints/tests and status/error tables.

## Data source
The app reads bundled JSON from:
- `src/data/summary.json`

This file is generated by `analysis-cli`.

## Install and run
```bash
cd dashboard
npm install
npm run dev
```

Build for production:
```bash
npm run build
npm run preview
```

## Refresh data
From repo root:
```bash
npm run analyze
```

That command writes new summary output into `dashboard/src/data/summary.json`.
